# Compilation Guide for Wii Menu TrimUI\n\n## Quick Start\n\n### 1. Install Dependencies\n\n**Debian/Ubuntu:**\n```bash\nsudo apt-get update\nsudo apt-get install -y \\\n  libsdl2-dev \\\n  libsdl2-image-dev \\\n  libsdl2-mixer-dev \\\n  libsdl2-ttf-dev \\\n  build-essential \\\n  pkg-config\n```\n\n**Fedora/RHEL:**\n```bash\nsudo dnf install -y \\\n  SDL2-devel \\\n  SDL2_image-devel \\\n  SDL2_mixer-devel \\\n  SDL2_ttf-devel \\\n  gcc \\\n  make\n```\n\n**Arch Linux:**\n```bash\nsudo pacman -S sdl2 sdl2_image sdl2_mixer sdl2_ttf base-devel\n```\n\n### 2. Clone and Compile\n\n```bash\ncd ~/wii-frontend\n\n# Using the build script (recommended)\nbash build.sh\n\n# Or manually\nmake clean\nmake\n```\n\n### 3. Run\n\n```bash\n./bin/wii-frontend\n```\n\n## Building for TrimUI Brick (Cross-compilation)\n\nIf building on a different architecture (e.g., building on x86_64 for ARM):\n\n```bash\n# Set cross-compiler\nexport CC=arm-linux-gnueabihf-gcc\nexport CFLAGS=\"-Wall -Wextra -O2 -std=c99 -fPIC -Isrc --sysroot=/path/to/trimui/sysroot\"\n\nmake clean\nmake\n```\n\n## Troubleshooting\n\n### SDL2 headers not found\n```\nfatal error: SDL2/SDL.h: No such file or directory\n```\n\n**Solution:**\n```bash\n# Find SDL2 location\npkg-config --cflags sdl2\n\n# Verify installation\npkg-config --modversion sdl2\n```\n\n### Missing libraries at runtime\n```\nerror while loading shared libraries: libSDL2.so.2.0\n```\n\n**Solution:**\n```bash\n# Find library location\nldconfig -p | grep SDL2\n\n# Or install: libsdl2-2.0-0 (debian/ubuntu)\nsudo apt-get install libsdl2-2.0-0\n```\n\n### Compilation flags\n\nIf you need to adjust compiler flags, edit `wii-frontend/Makefile`:\n\n```makefile\nCCFLAGS := -Wall -Wextra -O2 -std=c99 -fPIC -Isrc\n# Add optimization levels: -O0 (debug), -O3 (aggressive)\n# Add debugging: -g\n# Example:\nCCFLAGS := -Wall -Wextra -O2 -std=c99 -fPIC -Isrc -g\n```\n\n## Build Output\n\nSuccessful build produces:\n- `wii-frontend/bin/wii-frontend` - Executable binary\n- `wii-frontend/build/` - Object files (can be cleaned with `make clean`)\n\n## Development\n\n### Incremental builds\n```bash\nmake               # Compile only changed files\nmake clean && make # Full rebuild\n```\n\n### With debugging symbols\n```bash\n# Modify Makefile\nCFLAGS := -Wall -Wextra -g -std=c99 -fPIC -Isrc\nmake clean && make\n\n# Debug with gdb\ngdb ./bin/wii-frontend\n```\n\n## Dependencies Summary\n\n| Library | Version | Purpose |\n|---------|---------|----------|\n| SDL2 | 2.0+ | Window and rendering |\n| SDL2_image | 2.0+ | Image loading (PNG, JPG) |\n| SDL2_mixer | 2.0+ | Audio playback |\n| SDL2_ttf | 2.0+ | Font rendering |\n| libc | glibc 2.27+ | C standard library |\n| libm | - | Math library |\n\n## Next Steps\n\nAfter successful compilation:\n1. Configure channels: Edit `config/channels.conf`\n2. Add game icons: Place PNG files in `icons/`\n3. Add banners: Place PNG files in `banners/`\n4. Customize theme: Edit `assets/themes/default/theme.conf`\n5. Run: `./bin/wii-frontend`\n"
